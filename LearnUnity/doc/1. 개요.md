# 개요 

Unity 매뉴얼을 읽으면서 요약 정리한다. 매뉴얼을 읽으면 유니티를 알게 되는가? 그렇지는 않다. 왜냐하면 유니티는 도구이고 도구는 써야 용도를 제대로 알기 때문이다. 그래도 의미는 있는데 이미 경험이 있기 때문이다. 

과제로 만들고 하나씩 해결해 보는 경험도 중요하다. 대체로 최적의 솔루션인지가 중요하다.
